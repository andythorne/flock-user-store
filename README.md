# Flock User Store

Storage system for users with oauth2 support, written in Symfony 3.3 with FOSOAuth, FOSUser and EasyAdmin.

## Getting started

```bash
git clone https://github.com/andythorne/flock-user-store flock
cd flock

docker-compose up -d
```
